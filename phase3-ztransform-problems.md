# Phase 3 — Session 1: Z-transform 정의와 기본 성질

> 목표: 이미 아는 DTFT(이산 푸리에 변환)에서 출발해, Laplace가 FT에 했던 것과 **똑같은 방식**으로
> Z-transform을 유도한다. "극점이 단위원 안 = 안정"이라는 대응표까지 손으로 세운다.
> 선행: phase1-laplace-problems.md (W0~L5 — 노브 추가, ROC, pole 개념), 신호및시스템 수업(DTFT는 이미 앎)

## 도구 정리 (틀만 — 채우는 건 문제에서)

**복습**: Phase 1에서 FT → Laplace는 "회전 프로브 e^(−jωt)에 감쇠 노브 e^(−σt) 추가"였다.
`s = σ + jω`, σ=0(허수축)이면 FT로 복귀.

**DTFT (이미 아는 것)**:

```
X(e^(jω)) = Σ_{n=0}^{∞} x[n]·e^(−jωn)
```

(연속시간 e^(−jωt) 대신 이산시간 e^(−jωn) — 회전 프로브는 똑같이 있음)

**Z-transform**: 여기에도 Laplace와 같은 방식으로 노브를 하나 추가한다. 다만 이산시간에서는
"빼는 것"(−σt처럼 지수에 빼기)이 아니라 **곱하는 크기(magnitude)**로 노브가 들어간다:

```
z := r·e^(jω)   (r은 양의 실수, "크기" 노브)
X(z) = Σ_{n=0}^{∞} x[n]·z^(−n)
```

**왜 크기 노브인가**: z^(−n) = r^(−n)·e^(−jωn). r=1이면 e^(−jωn)만 남아 DTFT로 복귀 — 즉
**|z|=1 (복소평면의 단위원)이 Laplace의 허수축(σ=0)과 같은 역할**을 한다.
r≠1이면 r^(−n)이 n이 커질수록 커지거나(r<1) 작아져서(r>1), 발산하던 수열의 합도 수렴시킬 수 있다.

**ROC**: 합이 수렴하는 |z|(=r)의 범위. Laplace의 ROC(Re(s)의 범위)와 정확히 대응.

> 표기: ✍️ = 네가 쓴 답 그대로 / 💬 = Claude가 덧붙인 해설·교정

## 문제

### ⚠️ W0. 세 변환 비교 — 개념 설명 두 차례 필요했으나 최종 정리는 자력

- CTFT(연속 푸리에): X(jω) = ∫ x(t)e^(−jωt)dt
- Laplace: X(s) = ∫₀^∞ x(t)e^(−st)dt, s=σ+jω
- DTFT: X(e^(jω)) = Σ x[n]e^(−jωn)
- Z-transform: X(z) = Σ x[n]z^(−n), z=r·e^(jω)

- 처음엔 "잘 모르겠는데, 왜 z transform에선 e 표현을 z로 뭉개버린거야" → 💬 해설: z=r·e^(jω)는 e를 없앤 게 아니라 **e를 안에 품은 것** (z^(−n)=r^(−n)·e^(−jωn)). 이산 스텝에서 "매 스텝 비율 r 감쇠"는 애초에 거듭제곱(rⁿ) 구조라 자연스럽게 그렇게 나옴. 엄밀히는 샘플링 이론에서 z:=e^(sT)로 정의됨
- "뭐가 더하기 곱하기로 보인다는거야?" → 💬 재해설: s=σ+jω(덧셈으로 결합) vs z=r·e^(jω)(곱셈으로 결합) 구체적 대조
- 처음 답 "더했으니까 더하기고 곱했으니까 곱하기잖아" → 💬 동어반복 지적, **복리 비유**(연속복리 e^(rt) vs 이산복리 (1+r)ⁿ)로 재질문
- ✍️ 최종: **"연속은 지속적이고, 이산은 스텝마다 곱해서"** ← 자력으로 핵심(rate는 적분되어 누적→더하기, ratio는 반복되어 누적→곱하기) 정리 성공

### ✅ Z1. x[n] = aⁿ·u[n] (0<a<1) — 전부 자력

- ✍️ r = a/z 로 치환 → X(z) = z/(z−a) ← 등비급수 공식 적용 정확
- ✍️ pole = a, ROC = |z|>|a| ← 자력
- ✍️ "0.5ⁿ은 n이 커지면? → 0으로가" / "pole 단위원 안 = ? → 수렴한다?"(💬 조금 더 구체화 요청) → ✍️ 최종 "안정" 도달 — **pole 단위원 안 ⟺ 시간영역 감쇠 ⟺ 안정** 대응표 완성

### ✅ Z2. x[n] = u[n] — 전부 자력, 단 진동 여부는 해설 필요

- ✍️ X(z) = z/(z−1), pole=1, ROC=|z|>1 ← 자력 (Z1의 a=1 특수케이스로 정확히 재사용)
- ✍️ "단위원 위/안/밖?" → "딱 걸쳐있어. 그럼 계속 진동?" → 💬 **❌** 정정: 단위원 위=크기(magnitude) 얘기, 진동 여부는 각도(angle) 얘기. z=1은 각도 0 → 상수(constant), 진동 아님. 각도가 0이 아닌 단위원 위 pole이라야 진동(→ Z3 예고)

### ✅ Z3. x[n] = cos(ω₀n)·u[n] — 쪼개기·통분·pole 해석 전부 자력

- ✍️ 쪼개기: (1/2)z/(z−e^(jω₀)) + (1/2)z/(z−e^(−jω₀)) ← Z1 재사용, 자력
- ✍️ 통분: 분모 `z²−2z·cosω₀+1`, 분자 `z²−z·cosω₀` (Euler로 e^(jω₀)+e^(−jω₀)=2cosω₀, e^(jω₀)e^(−jω₀)=1 활용) ← 스텝별 힌트 후 최종 통합식 자력 도달
- ✍️ pole = e^(±jω₀), 단위원 **위**, 각도 ±ω₀ ← 자력. Z2(각도0=상수)와 대조되는 "각도≠0=영원한 진동" 케이스로 정확히 구분

### ✅ Z4. Shift property — 결과·경계항 위치 설명 전부 자력

- ✍️ Z{x[n−1]} = X(z)/z ← 자력 (Laplace 미분성질의 이산 대응)
- ✍️ x[−1]=0이 필요한 이유: "n=0일때 n-1=-1이라 그때 나와" ← 정확, 치환 시 경계항 위치를 짚어냄

### ⚠️ Z5. 전달함수 H(z) — 방향 실수 한 번, 자가교정 후 완료

`y[n] − 0.5y[n−1] = x[n]`, H(z) 구하기.

- ✍️ 1차 답: "H(z) = (2z−1)/2z" → 💬 **❌** 이건 X(z)/Y(z) (=1/H(z))였음, H(z)=Y(z)/X(z) 방향과 뒤바뀜
- ✍️ 자가교정: "아 맞다 실수로적었어" → **H(z) = 2z/(2z−1) = z/(z−0.5)** ← 정정 후 자력
- ✍️ pole=0.5, "안에있어서 안정해" ← 자력, 최종 결론 정확

## Python 검증 (다 푼 뒤)

```python
import numpy as np
from scipy import signal
import matplotlib.pyplot as plt

# Z1, Z5의 pole을 단위원과 함께 그려서 안/밖 확인
for label, num, den in [('Z1 (a=0.5)', [1], [1, -0.5]),
                         ('Z5 (H(z))', [1], [1, -0.5])]:
    z, p, k = signal.tf2zpk(num, den)
    print(label, 'poles:', p, '|pole|:', np.abs(p))

theta = np.linspace(0, 2*np.pi, 200)
plt.plot(np.cos(theta), np.sin(theta), 'k--', label='unit circle')
plt.scatter([0.5], [0], color='red', label='pole (a=0.5)')
plt.axis('equal'); plt.legend(); plt.grid(True, alpha=0.4)
plt.title('Z-plane: pole vs unit circle')
plt.show()
```

|pole| < 1이면 단위원 안 → 안정, 손으로 구한 ROC과 일치하는지 확인할 것.
