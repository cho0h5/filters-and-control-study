# Phase 1 — Session 1: Laplace transform 정의와 기본 성질

> 목표: 정의 적분에서 주요 transform pair를 **손으로** 유도하고, "미분 → s곱" 성질로
> Phase 0에서 풀던 ODE를 대수 문제로 바꿔 풀어보기
> 예상 시간: 3~4시간 (한 번에 다 안 해도 됨)

## 도구 정리

**정의** (unilateral, 공학 관례):

```
X(s) = ∫₀^∞ x(t)·e^(−st) dt,   s = σ + jω
```

**FT와의 관계**: e^(−st) = e^(−σt)·e^(−jωt) — FT의 회전 프로브 e^(−jωt)에
**감쇠 노브 e^(−σt)가 추가**된 것. σ = 0으로 놓으면(s = jω) FT로 돌아간다.
→ X(jω) 표기의 정체: X(s)를 허수축 위에서만 본 것.

**왜 감쇠 노브가 필요한가**: u(t)나 e^(+t)처럼 FT 적분이 발산하는 신호도,
e^(−σt)를 충분히 세게 곱하면 적분이 수렴한다. 수렴하는 σ의 범위 = **ROC** (region of convergence).

**적분 팁**: 이 세션의 모든 적분은 결국 `∫₀^∞ e^(−ct) dt = 1/c` (Re(c) > 0) 하나로 환원된다.

**미분 성질** (이 세션의 최종 병기):

```
L{ẋ(t)} = s·X(s) − x(0)
```

미분이라는 해석학 연산이 "s 곱하기"라는 대수 연산으로 바뀐다. 초기조건이 공짜로 딸려 들어오는 것에 주목.

> 표기: ✍️ = 내가 쓴 답 그대로 / 💬 = Claude가 덧붙인 해설·교정

## 문제

### ✅ W0. (워밍업) FT 정의 쓰기 — 자력

- ✍️ X(f) = ∫_{−∞}^{∞} x(t)e^(−jωt)dt ← 적분 구조 정확
- 💬 표기 교정: X(f)는 e^(−j2πft)와 짝, e^(−jωt)는 **X(jω)**와 짝. 이후 ω 표기로 통일
- ✍️ 바뀐 곳 ①: "t 앞이 허수부만 있었는데 복소수가 됐어" (−jωt → −st)
  - 💬 e^(−st) = e^(−σt)·e^(−jωt), 회전 프로브에 **감쇠 노브 추가**. 발산하는 신호도 수렴시킴 → ROC 개념의 출발
  - 💬 X(jω) 표기의 정체 해소: X(s)를 σ=0(허수축)에서만 본 것 = FT
- ✍️ 바뀐 곳 ②: "구간이 0~∞가 됐어"
  - 💬 이유: 제어 문제는 "t=0에 시스템 켬 + 초기조건"으로 시작 → 과거를 버림. L4에서 x(0)이 공짜로 나오는 복선

### ⚠️ L1. x(t) = e^(−at)·u(t)의 Laplace transform — 유도·pole 자력, ROC ❌

가장 중요한 pair. 이후 모든 것이 이것의 변형이다.

- ✍️ X(s) = 1/(a + s) ← 자력 유도
- ✍️ ROC: "a + s ≠ 0" → 💬 **❌**. ≠ 0은 "pole만 피하면 된다"는 뜻이 되는데,
  적분 ∫₀^∞ e^(−(s+a)t)dt가 수렴하려면 지수가 **감쇠해야** 한다: **Re(s + a) > 0, 즉 Re(s) > −a**
  - Re(s+a) < 0이면 발산, **= 0이어도** e^(jθt)로 영원히 회전만 해서 수렴 안 함
  - 그림: ROC = 복소평면에서 **pole −a의 오른쪽 반평면 전체**
  - a > 0(감쇠 신호)이면 ROC이 허수축을 포함 → σ=0 대입 가능 → **FT가 존재**하는 신호
- ✍️ pole = −a. "신호의 감쇠율이 pole 위치로 각인된다"
- ✍️ 후속 질문 3연타: "왜 Re(s+a)>0 ⟺ Re(s)>−a?" / "왜 실수부만 봐?" / "a는 실수야 복소수야?"
  - 💬 Re는 덧셈에 쪼개짐: Re(s+a) = Re(s) + a (a 실수일 때) → 보통 이항
  - 💬 수렴은 **크기** 문제: e^(−ct) = e^(−σ't)·e^(−jω't), 허수부는 |·|=1 회전만 → 실수부만이 감쇠 결정
  - 💬 a는 문제에선 실수지만 유도 어디서도 실수성을 안 씀 → 복소 a에도 성립, ROC: Re(s) > −Re(a). **L3에서 a = ∓jω₀로 재사용되는 복선**

### ✅ L2. x(t) = u(t) (unit step) — 계산 전부 자력, 해석은 해설 필요

- ✍️ 정의 적분 유도: 1/s
- ✍️ a = 0 교차검증: "똑같아"
- ✍️ ROC: Re(s) > 0 ← **L1 설욕 성공** (안 죽는 신호 → 노브를 조금이라도 틀어야 수렴)
- ✍️ pole = s = 0. 해석("경계 신호와 pole 위치의 관계")은 "모르겠는데" → 💬 해설:
  - s = 0은 원점 = **허수축 위**. 좌반평면=감쇠 / 우반평면=발산 / 허수축=경계라는 대응표에서,
    감쇠도 발산도 안 하는 u(t)의 pole이 정확히 경계선 위에 찍힌 것
  - L1 관점: u(t) = e^(−0·t)u(t), 감쇠율 0 → pole = −0 = 0. "감쇠율이 좌표로 각인"의 재확인
  - 복습 포인트: **pole 위치 → 시간 행동 읽기** (수식→해석 방향이 아직 약함, Phase 0 A-4와 같은 패턴)

### ⚠️ L3. x(t) = cos(ω₀t)·u(t) — Euler 쪼개기·pole 자력, 통분 ❌·해석 해설

적분을 새로 하지 말 것. **Euler 공식으로 e^(jω₀t)와 e^(−jω₀t)의 합으로 쪼갠 뒤 L1을 재사용**
(L1의 a에 복소수를 넣어도 성립한다).

- ✍️ 쪼개기: (1/2)/(s−jω₀) + (1/2)/(s+jω₀) ← 자력 (복소 a 복선 회수 성공)
- ✍️ 통분: "= 1/2(s²+ω²)" → 💬 **❌**. 분자를 1/2+1/2로 그냥 더함 — 통분은 **각 분자에 상대 분모를 교차곱**:
  - 분자 = (s+jω₀) + (s−jω₀) = **2s** (jω₀ 상쇄 — 실수 신호라 실수 계수로 떨어짐)
  - 분모 = s² − (jω₀)² = **s² + ω₀²** (j² = −1로 부호 반전 주의)
  - 표준형: **L{cos(ω₀t)u(t)} = s/(s² + ω₀²)**
  - 자가검증: 통분 후 분모=0 ⟺ s = ±jω₀, 통분 전 pole과 일치해야 함
- ✍️ pole = ±jω₀. 해석은 "σ=0, ω=?"까지 자력, 나머지 💬 해설:
  - σ = 0 → 감쇠 없음 = "영원히" / ω = ±ω₀ → "진동수 ω₀" — 이력서가 좌표에 각인
  - ±쌍인 이유: 실수 신호의 진동 = 회전 두 개의 합 → 복소 pole은 항상 **켤레쌍**
- 복습 포인트: **분수 통분 절차** (분자 교차곱), j² 부호 처리

### ⚠️ L4. 미분 성질 L{ẋ} = sX(s) − x(0) 유도 — 결과 도달, 경계항 논증·개념은 해설

- ✍️ 부분적분 전개 → L{ẋ} = sL{x} − x(0) ← 결과 도달, x(0)이 경계항에서 나오는 것 확인
- 💬 전체 전개 기록: ∫ẋe^(−st)dt = [e^(−st)x(t)]₀^∞ + s∫xe^(−st)dt = (0 − x(0)) + sX(s)
- ✍️ ∞쪽 경계항이 죽는 이유: "구간이 0~∞였으니 inf는 0이 되고" → 💬 **절반**. 경계항은 e^(−st) 혼자가 아니라
  **e^(−st)·x(t) 곱** — x가 버티거나 자라는 신호면 안 죽을 수 있음. 죽는 진짜 이유는 **ROC 안의 s만 쓰기 때문**:
  감쇠 노브가 x를 이기는 s에서는 정의상 곱이 0으로 감. ROC이 유도의 발판으로 실전 사용된 첫 장면
- ✍️ 개념: "모르겠는데" → 💬 해설: Phase 0 방식은 2단계(일반해의 C를 들고 다니다 맨 마지막에 IC로 확정 —
  재검증에서 흔들렸던 바로 그 절차). Laplace는 변환 첫 줄에 y(0)이 **이미 대수에 포함** → C 확정 단계가
  구조적으로 소멸. unilateral 적분의 보상 (W0 복선 회수)
- ✍️ 후속 질문: "L{}와 X()의 차이?" → 💬 동사와 명사: L{·}=연산자(행위), X(s)=결과물(이름).
  관례: 소문자=시간영역 ↔ 대문자=s영역 (x(t)↔X(s), y(t)↔Y(s))

### ✅ L5. 최종 병기 실전: ẏ + 2y = 0, y(0) = 3 — 힌트 1회 후 자력, pole 해석은 해설

Phase 0 Session A에서 손으로 풀던 유형. **답을 이미 아는 문제**로 새 도구를 검증한다.
시작: L{ẏ + 2y} = L{0}, 좌변에 L4 성질과 선형성(L{ẏ+2y} = L{ẏ} + 2L{y}) 적용.

- ✍️ 대수방정식: sY(s) − y(0) = −2L{y} + L{0}까지 자력, 여기서 막힘 →
  💬 힌트 1회: **L{y} = Y(s) (같은 것의 두 표기), L{0} = 0, y(0) = 3 대입** → sY − 3 = −2Y
- ✍️ Y(s) = 3/(s+2) ← 힌트 후 자력 (표기 교정: 괄호 필수, 3/s+2는 (3/s)+2로 읽힘)
- ✍️ y(t) = 3e^(−2t)u(t) ← L1 pair 역읽기 자력
- ✍️ Phase 0 방식과 일치 확인: "맞음"
- ✍️ pole 대응: "s = −2" — 위치는 맞음, 해석은 💬 해설 (L2·L3과 같은 패턴, 3회째):
  - **pole의 실수부 = y(t)의 지수** (−2 ↔ e^(−2t)). 좌반평면 깊을수록 빠른 감쇠, 허수축에 붙을수록 느림
  - inverse 없이 pole만 보고 시간 행동을 읽는 것 — Phase 1 3단계(대응표)의 요지
- 💬 음미: 미분방정식을 한 번도 "풀지" 않고 변환+이항+나누기만으로 해에 도달. IC는 첫 줄에 자동 포함(L4 보상)

## Python 검증 (다 푼 뒤)

```python
import sympy as sp

t, s = sp.symbols('t s', positive=True)
a, w0 = sp.symbols('a omega0', positive=True)

# L1~L3 손계산 확인
for name, x in [('L1', sp.exp(-a*t)),
                ('L2', sp.Heaviside(t)),
                ('L3', sp.cos(w0*t))]:
    X = sp.laplace_transform(x, t, s, noconds=True)
    print(name, '→', X)

# L5: ODE를 sympy로 직접 풀어서 y(t) 비교
y = sp.Function('y')
sol = sp.dsolve(sp.Eq(y(t).diff(t) + 2*y(t), 0), y(t), ics={y(0): 3})
print('L5 →', sol)
```
