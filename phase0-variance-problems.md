# Phase 0 — Session C: 분산 대수 연습문제

> 목표: Var(aX+b) = a²Var(X)를 외우지 않고 정의에서 꺼내 쓰기
> (재검증 Var(3X+5) = 41 ❌의 설욕전)
> 표기: ✍️ = 내가 쓴 답 그대로 / 💬 = Claude가 덧붙인 해설·교정

## 도구 정리

정의: `Var(X) = E[(X − μ)²]` — 편차 제곱의 평균 = 퍼짐

| 규칙 | 식 | 이유 (정의에서) |
|---|---|---|
| 밀기는 공짜 | Var(X+b) = Var(X) | 평균도 같이 밀려서 편차 불변 |
| 늘리기는 제곱 | Var(aX) = a²Var(X) | 편차가 a배 → 제곱하니 a² |
| 독립이면 합산 | Var(X+Y) = Var(X)+Var(Y) | 일반식엔 +2Cov(X,Y)가 있는데 독립이면 0 |

대조: 기대값은 그냥 통과 — E[aX+b] = aE[X] + b
참고: standard deviation으로 보면 σ(aX) = |a|σ(X). 분산의 제곱은 단위가 제곱이라서.

## 문제

### ✅ C1. (설욕전) Var(X) = 4일 때 Var(3X + 5) = ? — 자력, 설욕 성공 🎉

- ✍️ 답: 36
- 💬 재검증 ❌(41)이 교정됨: 3²·4 = 36, +5는 밀기라 공짜

### ✅ C2. Var(−X) = ? — 자력

- ✍️ 같음. 뒤집어지기만 하고 값들이 서로 벌어진 정도는 같음
- 💬 정의 기반 사고 정확. 수식으론 (−1)² = 1

### ✅ C3. X, Y independent, Var(X) = 1, Var(Y) = 4. Var(2X − Y + 7) = ? — 자력

- ✍️ 답: 8
- 💬 함정 통과: 2²·1 + (−1)²·4 = 8, +7 버림

### ✅ C4. (하이라이트) 독립 측정 N개(각 분산 σ²)의 평균 (X₁+...+X_N)/N 의 분산은? — 자력

- ✍️ 답: (σ² + ... + σ²) / N^2 = Nσ² / N^2 = σ² / N
- ✍️ 의미: 여러번 측정할수록 분산이 작아진다
- 💬 세 가지 의미: ① 이동평균 필터(B1)가 노이즈를 죽이는 이유 — "평균=분산 1/N"(확률 언어) = "평균 필터=low-pass"(주파수 언어), 같은 사실 ② std는 σ/√N — 노이즈 절반 줄이려면 측정 4배 (수확 체감) ③ 칼만 필터의 출발 질문: "측정마다 품질이 다르면? 과거 추정과 새 측정을 합칠 땐?" → 최적 가중 평균을 재귀로 계산하는 것이 칼만(Phase 6)

### ✅ C5. (보너스) Y = X일 때 Var(X+Y)를 규칙 2로 정확히 계산하면? 규칙 3을 (잘못) 적용하면? — 자력

- ✍️ 정확한 값: 4Var(X) / 잘못 적용한 값: 2Var(X) / 차이의 정체: Covariance
- 💬 차이 2Var(X) = 2·Cov(X,X). "같이 움직이는 변수의 합은 독립일 때보다 퍼짐이 크다"의 극단 사례
